# Cuba Car Wash

Landing page premium para **Cuba Car Wash** — detailing profesional en Las Vegas, NV.

## Stack

- Next.js 16 + Tailwind CSS
- Google Imagen 4.0 (imágenes generadas con IA)
- Animaciones tipo Apple (scroll reveal, parallax, hover glow)

## Secciones

- Hero cinematográfico con parallax
- 6 servicios con imágenes IA
- Antes / Después
- Proceso en 3 pasos
- Sección Las Vegas
- Identidad cubana
- Sobre Jorge Machado
- Galería premium
- Testimonios
- CTA con WhatsApp + Instagram
- Sticky CTA mobile

## Correr localmente

```bash
npm install
npm run dev
```

## Variables de entorno

Crear `.env.local`:

```
GOOGLE_AI_API_KEY=tu_api_key
```
